## Overview

GuardianShield ("we," "our," or "us") operates a Web3 threat intelligence and autonomous security platform. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our services.

**Geographic Scope:** This service is available only to users located within the United States.

---

## Information We Collect

### 1. Personal Information
- **Account Information:** Name, email address, phone number
- **Identity Verification:** KYC/AML documentation as required
- **Payment Information:** Wallet addresses, transaction hashes (no private keys)
- **Communication Records:** Support tickets, feedback, correspondence

### 2. Technical Information
- **Usage Data:** API calls, feature usage, access patterns
- **Device Information:** IP addresses (US only), browser type, device identifiers
- **Blockchain Data:** Public wallet addresses, transaction history, smart contract interactions
- **Threat Intelligence:** Security event data, risk scores, behavioral analytics

### 3. Automatically Collected Information
- **Log Files:** Server logs, error reports, performance metrics
- **Cookies:** Session management, preferences, analytics
- **Security Monitoring:** Login attempts, suspicious activities, fraud prevention

- **Service Delivery:** Provide threat intelligence, security monitoring, and platform access
